# Crawlspace CLI

`crsp` is a command-line interface (CLI) for creating, running, and deploying web crawlers to [crawlspace.dev](https://crawlspace.dev).

## Installation

```
npm install -g crsp
```

## Usage

```
crsp [command] [options]
```

For more information on each command, use the `--help` flag:

```
crsp [command] --help
```
